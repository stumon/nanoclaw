# Stock Screener 量化选股

基于华尔街五种量化筛选框架，对个股做结构化评估或按框架搜索候选股票。

## 两种模式

### 模式 1：个股分析

用户给出股票代码或名称（如 "分析 AAPL"、"帮我看看特斯拉"），按以下步骤执行：

1. 用 WebSearch 搜索该股票的关键财务数据（见下方搜索词模板）
2. 用 FetchURL 抓取 Finviz 页面补充数据（可选）
3. 用五套筛子逐一评估
4. 输出结构化报告

### 模式 2：筛选搜索

用户指定一种筛选方法（如 "帮我找价值股"、"有哪些高成长股"），按以下步骤执行：

1. 根据用户指定的筛选类型，用 WebSearch 搜索符合条件的股票
2. 找出 3-5 只候选股票
3. 对每只候选股简要列出关键指标
4. 给出筛选结论

## 五套筛选框架

### 一、价值筛选 (Value Screen)

找被市场低估的公司。

| 指标 | 阈值 | 含义 |
|------|------|------|
| P/E | 低于行业中位数（粗略起点 <15x） | 市场估值偏低 |
| EV/EBITDA | 低于历史均值（粗略起点 <10x） | 企业价值相对便宜 |
| FCF Yield | >5% | 自由现金流回报率高 |
| P/B | <1.5x | 股价接近净资产 |
| 内部人买入 | 90天内有管理层买入 | 真金白银投票 |
| 股息率 | 高于市场均值（粗略起点 >2%） | 现金回报实在 |

判断标准：6 项中 PASS 4 项以上为通过。
关键提醒：便宜不等于好，需排除价值陷阱（基本面持续恶化的公司）。最好的价值机会 = 好公司 + 临时利空 + 市场过度反应。

### 二、成长筛选 (Growth Screen)

找增长最快且增长在加速的公司。

| 指标 | 阈值 | 含义 |
|------|------|------|
| 收入增长 | >15% YoY | 增长够快 |
| 盈利增长 | >20% YoY | 利润增长比收入更快 |
| 收入加速度 | 连续 2 季度 YoY 增速提升 | 越来越快 |
| 利润率 | 毛利率或 EBITDA 利润率 YoY 提升 | 规模效应 |
| ROIC | >15% | 资本回报率高，增长有质量 |
| NRR（SaaS） | >110% | 老客户花更多钱 |

判断标准：5 项中 PASS 3 项以上为通过（NRR 仅适用于 SaaS 公司）。
关键提醒：避免为高增长付太高价格。Rule of 40（增长率% + 利润率% > 40）用于平衡增长和盈利。

### 三、质量筛选 (Quality Screen)

找长期稳定、财务健康的好公司。

| 指标 | 阈值 | 含义 |
|------|------|------|
| 持续增长 | 5 年以上 | 不是一锤子买卖 |
| 利润率波动 | 5 年内波动 <5 个百分点 | 经得起周期考验 |
| ROE | >15% | 股东资本回报高 |
| 负债率 | Debt/Equity <1.0 | 不靠杠杆撑业绩 |
| FCF 转化率 | FCF/净利润 >80% | 利润是真金白银 |
| 内部人持股 | >5% | 管理层利益绑定 |

判断标准：6 项中 PASS 4 项以上为通过。
关键提醒：这是巴菲特 "以合理价格买入优秀公司" 的量化版。单年不惊艳，5-10 年复利惊人。

### 四、做空筛选 (Short Screen)

找正在恶化但市场还没反应过来的公司。对个人投资者最大价值是避雷。

| 信号 | 量化方式 | 含义 |
|------|----------|------|
| 收入下滑 | YoY 下滑或连续 2 季增速放缓 | 基本面在变差 |
| 利润率压缩 | 毛利率或营业利润率 YoY 下降 >2ppt | 竞争加剧或成本失控 |
| 应收/库存异常 | 增速超过收入增速 1.5 倍 | 收入质量存疑 |
| 内部人卖出 | 90天内 C-suite 累计卖出 >$1M | 管理层在跑路 |
| 估值溢价 | P/E 或 EV/EBITDA 高于行业中位数 50%+ | 市场在做梦 |
| 高空仓 | Short Interest >10% of float | 聪明钱已在做空 |
| 会计红旗 | 换审计师、重述财报、审计意见非标准 | 最危险的信号 |

判断标准：触发 3 个以上信号为高风险。
关键提醒：如果你持有的股票触发多个做空信号，认真审视是否该卖。盯住应收账款和库存，收入增 10% 但应收增 30% 可能是塞货。

### 五、特殊事件筛选 (Special Situation Screen)

找因公司结构变化而被错误定价的机会。

| 事件 | 为什么有机会 | 发现方式 |
|------|------------|----------|
| IPO/SPAC 锁定期到期 | 抛压砸出低价但基本面没变 | IPO 招股书 S-1，通常上市后 90-180 天 |
| 分拆 Spin-off | 被动卖盘 + 市场不了解新公司 | SEC EDGAR Form 10-12B |
| 重组出表 | 坏消息出尽，轻装上阵 | SEC EDGAR 8-K，关键词 restructuring |
| 激进投资者介入 | 外部力量推动改善 | SEC EDGAR 13D 文件（持股 >5%） |
| 管理层换血 | 新管理层大扫除后超预期 | SEC EDGAR 8-K Item 5.02 |

判断标准：存在事件 + 基本面未恶化 = 值得深入研究。
关键提醒：彼得-林奇特别喜欢 Spin-off 机会。Alpha 来源不是 "便宜" 而是 "市场结构性错误定价"。

## 数据获取方法

### WebSearch 搜索词模板

个股分析时，按以下模板搜索（将 TICKER 替换为实际代码）：

搜索 1（估值+基本面）：
```
TICKER stock P/E ratio EV/EBITDA P/B FCF yield 2025
```

搜索 2（增长数据）：
```
TICKER revenue growth earnings growth YoY quarterly 2025
```

搜索 3（质量+风险）：
```
TICKER ROE debt equity insider trading short interest
```

### FetchURL 数据源

Finviz 个股页面包含大部分所需指标：
```
FetchURL(url: "https://finviz.com/quote.ashx?t=TICKER")
```

从返回内容中提取：P/E、Forward P/E、PEG、P/B、P/S、EV/EBITDA、Profit Margin、ROE、ROI、Debt/Eq、Short Float、Insider Own、Dividend Yield 等。

## 风格组合推荐

当用户问 "我该用哪种筛选" 时，参考以下推荐：

| 风格 | 推荐筛选 | 原因 |
|------|----------|------|
| 稳健长期 | 质量 + 价值 | 好公司 + 好价格，犯错成本最低 |
| 追求高回报 | 成长 + 质量 | 高增长 + 高质量，避免为垃圾付高价 |
| 有经验想找超额收益 | 特殊事件 | 信息不对称最大，但需更多研究 |
| 保护现有持仓 | 做空（排雷） | 定期扫描持仓，及早发现恶化信号 |
